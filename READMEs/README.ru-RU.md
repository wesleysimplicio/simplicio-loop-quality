# 🛡️ simplicio-loop-quality — Проверяемое качество для любого репозитория

<p align="center"><img src="../docs/assets/quality-loop-hero.webp" alt="Светящийся цикл проверки" width="920" /></p>

simplicio-loop-quality — расширение качества Simplicio: оно выбирает применимые проверки, предоставляет специализированных агентов, проверяет доказательства и передаёт окончательное решение Completion Oracle из simplicio-loop. Второй оркестратор не создаётся.

- 36 fail-closed линий качества;
- 24 декларативных агента;
- доказательства привязаны к запуску, коммиту, политике и инструменту;
- завершение разрешает только simplicio-loop.
