# 🛡️ simplicio-loop-quality — あらゆるリポジトリの検証可能な品質

<p align="center"><img src="../docs/assets/quality-loop-hero.webp" alt="光る検証ループ" width="920" /></p>

simplicio-loop-quality は Simplicio の品質拡張です。適用すべき検査を決め、専門エージェントを提供し、証拠を監査して、最終判断を simplicio-loop の Completion Oracle に委ねます。別のオーケストレーターは作りません。

- fail-closed の品質レーン 36 個;
- 宣言的なエージェント 24 個;
- run、commit、ポリシー、ツールに結び付いた証拠;
- 完了を承認できるのは simplicio-loop のみ。
