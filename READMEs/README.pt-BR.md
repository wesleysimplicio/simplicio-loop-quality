# 🛡️ simplicio-loop-quality — Qualidade verificável para qualquer repositório

<p align="center"><img src="../docs/assets/quality-loop-hero.webp" alt="Loop luminoso de verificação" width="920" /></p>

O simplicio-loop-quality é a extensão de qualidade do Simplicio: define os checks aplicáveis, fornece agentes especializados, audita evidências e entrega a decisão final ao Completion Oracle do simplicio-loop. Ele não cria um segundo orquestrador.

- 36 lanes de qualidade fail-closed;
- 24 agentes declarativos;
- evidência vinculada a run, commit, política e ferramenta;
- conclusão somente pelo simplicio-loop.
