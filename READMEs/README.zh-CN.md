# 🛡️ simplicio-loop-quality — 面向任意仓库的可验证质量

simplicio-loop-quality 是 Simplicio 的质量扩展：决定适用的检查，提供专用代理，审计证据，并将最终决定交给 simplicio-loop 的 Completion Oracle。它不会创建第二个编排器。

- 36 条 fail-closed 质量通道；
- 24 个声明式代理；
- 证据绑定到运行、提交、策略和工具；
- 只有 simplicio-loop 可以批准完成。
