# 🛡️ Simplicio-loop-quality — 모든 저장소를 위한 검증 가능한 품질

<p align="center"><img src="../docs/assets/quality-loop-hero.webp" alt="빛나는 검증 루프" width="920" /></p>

Simplicio-loop-quality는 Simplicio의 품질 확장입니다. 적용 가능한 검사를 결정하고 전문 에이전트를 제공하며, 증거를 감사한 뒤 최종 결정을 simplicio-loop의 Completion Oracle에 맡깁니다. 별도의 오케스트레이터를 만들지 않습니다.

- fail-closed 품질 레인 36개;
- 선언적 에이전트 24개;
- run, commit, 정책, 도구에 바인딩된 증거;
- 완료 승인 권한은 simplicio-loop에만 있음.
