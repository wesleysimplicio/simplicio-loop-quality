# 🛡️ simplicio-loop-quality — Verifizierbare Qualität für jedes Repository

<p align="center"><img src="../docs/assets/quality-loop-hero.webp" alt="Leuchtender Verifikations-Loop" width="920" /></p>

simplicio-loop-quality ist die Qualitätserweiterung von Simplicio: Sie bestimmt anwendbare Checks, stellt spezialisierte Agents bereit, prüft Nachweise und übergibt die endgültige Entscheidung an den Completion Oracle von simplicio-loop. Es entsteht kein zweiter Orchestrator.

- 36 fail-closed-Qualitäts-Lanes;
- 24 deklarative Agents;
- Nachweise sind an Run, Commit, Policy und Tool gebunden;
- Abschluss nur durch simplicio-loop.
