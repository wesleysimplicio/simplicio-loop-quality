# Contributing

1. Read `AGENTS.md` and `docs/ARCHITECTURE.md`.
2. Work from one issue in `backlog/issues.json`.
3. Do not introduce any orchestration responsibility owned by `simplicio-loop`.
4. Add tests and commit-bound evidence for every acceptance criterion.
5. Run `python scripts/check.py` and the relevant integration/system gates.
6. Open a focused pull request and include the quality evidence paths.

No skipped, flaky, stale or unexecuted test may be presented as passing evidence.
